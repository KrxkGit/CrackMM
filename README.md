# ⚡CrackMM

本软件为课程视频播放器激活客户端。

## 📖使用方法

### 📜配置

首先启动本软件，然后配置播放器使用代理 

> 127.0.0.1:8080

### 📜代理模式

基于对课程知识产权的保护，软件默认将使用代理模式，即**直接转发激活请求**。**同时将请求保存到**

> store.keyFile

### 📜缓存模式

当文件

> store.keyFile

存在时，软件将直接返回该文件保存的请求。

### 📜安卓端方案

对于安卓端，由于安卓端播放器未提供配置代理功能，且安卓端播放器不走系统代理，故可通过类似

> Postern

的 **VPN** 工具设置 **全局 HTTP 代理服务器**（注意：VPN 开启后将导致其他应用无法上网，激活播放器后即可关闭 VPN，往后同系列课程无需再次激活）。
